Criação de Tabelas

Para criar tabelas, utiliza-se a tag "table" para iniciar a estrutura, em seguida utiliza-se a tag "tr" (table row, ou linha da tabela) para abrir as linhas e, dentro dela, utiliza-se a tag "td" (table data, ou dados da tabela), que formam as colunas referentes a linha da tabela, com os dados dentros. Para destacar os "títulos" da tabela, utiliza-se a tag "th" (table header) ao invés de "td"!
  • Exemplo:

    • Produtos Código Preço
    Notebook 001 4.999,00
    Celular 002 3.499,00

Para mesclar as células de uma tabela, pode-se utilizar dois atributos:
  • Colspan (Extensão de coluna): Mescla Colunas
  • Rowspan (Extensão de linha): Mescla Linhas

    • Exemplo:

      • Desconto Cupom 1
      Cupom 2

Criação de Formulários

Para criar formulários, utiliza-se a tag "form", e você pode usar a tag "input" para criar os botões ou campos de texto, para colocar as informações.
