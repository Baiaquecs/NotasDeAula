Maneiras de Utilizar CSS

Há 3 Maneiras De Utilizar CSS, são elas:
  • Inline Style
  • Internal Style Sheet
  • External Style Sheet (Mais Recomendado)

- Inline Style

É uma maneira de formatar arquivos HTML, utilizando o atributo "style" dentro das próprias tags HTML.

    •  Lorem ipsum dolor sit amet consectetur adipisicing elit. Labore repellat delectus, nemo, recusandae nulla fugiat quis natus iusto exercitationem,
     aut dignissimos fugit illo earum illum nostrum blanditiis est deserunt amet.

O principal problema dessa maneira de utilizar CSS é a falta de otimização, visto que se você quiser fazer uma mudança geral, teria que ir de tag em tag.

Class and ID

Uma outra alternativa aos seletores globais, como "p" ou "h1", que estilizam todas as tags a que eles se referem de maneira padrão, são as "Classes" (Class) e "Identidades" (ID).

  1. Class
    • Define um padrão que pode ser utilizado mais de uma vez, em diferentes tags.
  2. Id
    • Define um padrão que deve ser utilizado apenas uma vez em todo o código.
- Para criar uma classe, utiliza-se o mesmo formato dos seletores, com a diferença de iniciar com um ponto.
E para chamá-la, basta utilizar a tag "class" e escrever seu nome - não precisa do ponto.
  • Exemplo:
    •     .exemplo { }

        <p class="exemplo">
- Para criar uma id, também utiliza-se o mesmo formato dos seletores, porém inicia-se com uma hashtag.
E para chamá-la, basta utilizar a tag "id" e escrever seu nome - sem a hashtag.
  • Exemplo:
    •     #exemplo { }

        <p id="exemplo">

Div and Span

Essas duas tags são tags que "dividem" ou "separam" áreas do site - e são tags HTML.
  1. Div
    • Divide o site em blocos - é uma tag em bloco (ocupa 100% da largura da região delimitada, logo, duas tags "div" seguidas ficariam uma embaixo da outra no site).
  2. Span
    • Divide o site apenas na região selecionada - é uma tag in line (duas tags "span" seguidas ficariam uma do lado da outra, na mesma linha, no site, como em um texto)
- Uma Div utiliza-se da seguinte forma:
  • Exemplo:
    •     <div>Texto 1</div>    <div>Texto 2</div>   gerará:

      Texto 1
      Texto 2
- Já uma Span, utiliza-se da seguinte forma:
  • Exemplo:
    •     <span>Texto 1</span>    <span>Texto 2</span> gerará:

      Texto 1 Texto 2

Border

É uma propriedade CSS que define bordas para a área desejada. Os principais atributos das bordas, são:
  • border-color
  • border-width
  • border-style
Os principais border-style são:
  • solid, dotted, dashed, outset

Font

Font-family

Font-family é um atributo que define o estilo da fonte em si. Um ótimo site para utilização de fontes seguras e comuns a maioria dos usuários, usa-se o site: w3schools.com

Utiliza-se essa sintaxe para esse atributo em CSS:
  • { font-family: "family-name", generic-family, initial|inherit; }
O Browser tentará utilizar as primeiras fontes específicas, e caso elas não estejam disponíveis no navegador, é sempre importante deixar uma fonte genérica. Ele lerá as fontes pela ordem que estiverem colocadas.

Font-size

É um atributo CSS que define o tamanho das fontes. Suas principais unidades de medida são:
  • px: é um tamanho fixo, em pixels
  • %: é um tamanho relativo a janela do browser
  • em: é um tamanho relativo ao container pai

Font-weight

Atributo CSS que define o "peso" da fonte, o quão "grossa" ou "forte" ela será - muito utilizada para deixar o texto em negrito, por exemplo.
Exemplo:
  • font-weight: bold = texto em negrito
  • font-weight: normal = texto normal
Ainda há a opção de font-weight que varia de 100 à 900, que altera o quão negrito o texto fica. Exemplo:
  • font-weight: 100 = texto em weight 100
  • font-weight: 900 = texto em weight 900

Font-style

Esse atributo define alguns estilos de texto, como:
  • Italic: O texto fica em itálico
  • Oblique: É praticamente um itálico com outro nome
  • Normal: O texto fica normal

Text-decoration

Esse atributo CSS define algumas decorações para os textos. São elas:
  • Underline: Define uma linha abaixo do texto
  • Line-through: Define uma linha no meio do texto
  • Overline: Define uma linha acima do texto

Font

Nós podemos utilizar todos essas configurações de uma única vez apenas com o atributo Font - porém não acho que seja bom, pois fica desorganizado.

Background

Essa propriedade CSS define as características do plano de fundo do container. E alguns atributos são utilizados para definir suas características.

Background-color

Esse atributo altera a cor do plano de fundo. Por exemplo:
  • Plano de fundo azul
  • Plano de fundo vermelho

Background-image

Esse atributo CSS define uma imagem como plano de fundo, ou seja, que permanece "atrás" de todo conteúdo.

Background-repeat

Esse atributo define a forma como a "Background-Image" se repete no browser. Seus principais valores são:
  • background-repeat: no-repeat = imagem só aparece uma vez;
  • background-repeat: repeat-x = imagem se repete continuamente na horizontal;
  • background-repeat: repeat-y = imagem se repete continuamente na vertical;
  • background-repeat: repeat = imagem se repete tanto na vertical quanto na horizontal.

Background-attachment

Esse atributo CSS define a maneira como a imagem irá se comportar quando a página do navegador for rolada para baixo. Seus valores são:
  • background-attachment: fixed = imagem permanece "fixa" no navegador - sua posição não muda quando a página é rolada para baixo, ou seja, a imagem aparece sempre;
  • background-attachment: scroll = imagem permanece com uma posição fixa na página, e não no navegador - quando a página é rolada para baixo, a imagem "some".

Background-position

É um atributo CSS que alinha a imagem do background de acordo com as instruções. Ele pode receber dois valores, que são:
  • Primeiro valor: left ou center ou right - define sua posição no eixo horizontal;
  • Segundo valor: top ou center ou bottom - define sua posição no eixo vertical.

OBS

  • Essas propriedades são aplicadas de acordo com o container em que a imagem for inserida como background;
  • Assim como em Font, todas essas configurações podem ser aplicadas de uma única vez apenas com a propriedade "Background" - mas fica desorganizado;
  • É possível utilizar uma cor de fundo e uma imagem como Background ao mesmo tempo!

Modelo de Caixa

Os elementos HTML se comportam como se estivessem em "caixas", que podem ser do tipo inline, block ou inline-block - o que será explicado mais a frente.
Esse comportamento faz com que seja possível delimitar:

  • Padding: Espaçamento interno, entre o conteúdo e a "caixa";
  • Margin: Espaçamento externo, entre a "caixa" e outros elementos.

Essas duas propriedades podem ser ajustadas conforme a necessidade, utilizando "-left", "-right", "-top" ou "-bottom", logo após elas.
Pode-se também ajustá-las fornecendo 2 ou 4 valores:

  • 2 valores: o primeiro refere-se aos espaçamentos horizontais (right and left) e o segundo aos espaçamentos verticais (top and bottom);
  • 4 valores: eles alteram os espaçamentos nos 4 cantos da "caixa", começando no topo e seguindo o sentido horário.

Margin-Collapse

Esse é um comportamento que vale a pena destacar:
  • Quando nós temos dois elementos ("caixas") que estão perto um do outro e há margens que os separam, os valores não são somados, eles se sobrepõem, permanecendo a margem maior os separando.

OBS

Vale a pena se atentar ao fato de alguns seletores já possuírem margin pré-definidos, como os heading. Por isso é bom, ao inicar um projeto, utilizar o seletor universal " * " para zerar essas formatações padrão.

Elementos Flutuantes

Float

É um atributo CSS que cria um novo "fluxo flutuante" na página: literalmente faz o container flutuar. Essa propriedade é muito útil, mas deve-se tomar cuidado devido a alguns comportamentos que ela tem e causa na página e nos outros elementos.
Ela é muito utilizada para "alinhar" os containeres e/ou alterar sua posição. O float tem dois valores:

  • right: faz com que os elementos flutuem à direita;
  • left: faz com que os elementos flutuem à esquerda.
Alguns comportamentos dos elementos com float:
  • Se dois elementos estão no mesmo cointainer e os dois estão com float, eles se alinhem segundo a ordem de chamada (quem vem primeiro no código, é o 1°);
  • À priorios elementos flutuantes se alinham conforme o espaço no browser - diminuir ou aumentar a janela do browser pode alterar a posição aparente do elemento;
  • Os elementos fora do fluxo flutuante se comportam como se os elementos flutuantes não existissem - eis a importância da propriedade "clear";
  • Devido ao comportamento acima, os cointaineres a que os elementos flutuantes pertencem não se expandem para que eles "caibam" - mais uma vez a importância do "clear";
  • O elemento flutuante sempre tenta se alinhar o mais a direita ou a esquerda possível - é importante se atentar ao tamanho dos cointaineres

Clear

Esse atributo CSS faz com que o elemento "limpe" o fluxo flutuante - o elemento passa a "reconhecer" a existência do elemento flutuante e "joga" o container para baixo desses elementos - por isso é muito importante para a estética da página.
Ele possui 4 valores, que são:

  • none: não altera nada;
  • right: limpa o fluxo flutuante à direita;
  • left: limpa o fluxo flutuante à esquerda;
  • both: limpa o fluxo flutuante à direita e à esquerda.

OBS

Para utilizar background com elementos flutuantes, por exemplo, deve-se:

  1. Criar uma div principal;
  2. Definir o background;
  3. Colocar os elementos flutuantes em divs dentro da div principal
  4. Utilizar o atributo clear dentro da div principal - pode ser em uma outra div apenas com uma classe ".clear", por exemplo.

Elementos block, inline e inline-block

Os elementos HTML se comportam de maneira a definirem 3 grandes grupos, que são eles:

  • Elementos Block;
  • Elementos Inline;
  • Elementos Inline-Block.

Elementos Block

Os elementos do tipo Block são elementos que possuem uma largura - WIDTH - que ocupa todo o espaço da janela do browser.
Devido a esse comportamento, dois elementos Block chamados em sequência são posicionados um abaixo do outro. Exemplos desse tipo de elemento são:

  • <h1>
  • <p>
  • <table>

Elementos Inline

Os elementos do tipo Inline possuem a largura de acordo com o tamanho do seu conteúdo. Diferentemente dos elementos Block, se dois elementos Inline forem chamados em sequência, eles serão posicionados na mesma linha - se houver espaço. Exemplos de Elementos Inline:

  • <a>
  • <span>
  • <img>

Elementos Inline-Block

Os elementos Inline-Block possuem a largura de acordo com o tamanho do seu conteúdo - assim como os elementos Inline - mas, se dois elementos Inline-Block forem chamados em sequência e o segundo não couber por inteiro na linha, eles ficarão um embaixo do outro - como os elementos Block. Pode-se alterar essas configurações dos elementos utilizando o atributo "display":

  • display: block
  • display: inline
  • display: inline-block

Position

Esse atributo CSS define a forma como o atributo irá se posicionar na janela do browser e também o local. Pode receber quatro valores:

  • static;
  • relative;
  • absolute;
  • fixed.

Static

O posicionamento Static é o padrão já aplicado a todos os elementos em HTML.

Relative - de acordo com a posição original

O posicionamento Relative utiliza propriedades de deslocamento, para posicionar o objeto de acordo com a sua posição "original". Assim como com o Static, com o posicionamento Relative, o elemento sobe a medida que a página é rolada para baixo. Aos atributos para deslocamento definem-se valores que "afastam" o container da sua posição original, e os mais utilizados são (para valores positivos):

  • top: desloca o elemento para baixo;
  • bottom: desloca o elemento para cima;
  • right: desloca o elemento para esquerda;
  • left: desloca o elemento para direita;

Se nenhum valor for atribuído a esses atributos, o elemento permanece na posição "original".

Absolute - de acordo com o coitainer em que está inserido, se o coitainer possuir algum valor de posição

Com o posicionamento Absolute, o container sai do fluxo padrão do documento - parecido com um Elemento Flutuante -, se sobrepoõe aos outros elementos e a sua posição é defina com os mesmos atributos de deslocamento, porém, eles deslocam o objeto de acordo com o container em que o elemento está inserido, se ele tiver uma posição diferente da Static - senão ele usará o body como padrão. São utilizados os mesmos atributos que no Relative, mas que leva a um comportamento diferente, que são (para valores positivos):

  • top: afasta o elemento do topo do container;
  • bottom: afasta o elemento do canto inferior do container;
  • right: afasta o elemento do canto direito do container;
  • left: afasta o elemento do canto esquerdo do container.

O elemento sobe a medida que a página é rolada para baixo.

Fixed - fixo em relação a janela do navegador

O posicionamento Fixed utiliza como padrão a janela do browser e mantém o elemento fixo à ela - aparecendo mesmo se a página for rolada para baixo. Ele utiliza os mesmos atributos de posicionamento que o Absolute, porém sempre em relação ao <body>:

  • top: afasta o elemento do topo do body;
  • bottom: afasta o elemento do canto inferior do body;
  • right: afasta o elemento do canto direito do body;
  • left: afasta o elemento do canto esquerdo do body.

Sobreposição de Elementos

Ao utilizar as propriedades de posicionamento (Position: Relative, Absolute, Fixed) pode ocorrer que algum elemento se sobreponha a outros. Para definir, então qual elemento aparece primeiro em uma posição, utiliz-se a propriedade CSS:

  • z-index: quanto maior o número atribuído, mas a frente aparece o elemento.

Se for atribuído o mesmo valor a mais de um elemento, a sobreposição segue a ordem de chamada do código: os elementos que foram escritos primeiros ficam mais abaixo.

Formatação de Links

Para estilizar links do CSS, pode-se utilizar 4 atributos, que alteram o tipo de link que será estilizado. São eles:

  • a:link = define links que ainda não foram visitados pelo usuário;
  • a:visited = define links que já foram visitados pelo usuário;
  • a:hover = define links com o cursor do mouse em cima;
  • a:active = define links no momento em que são clicados pelo usuário.
